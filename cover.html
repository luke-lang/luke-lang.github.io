A customizable language
1. Custom language definition
$ = {
     print: {
         follow: ["{input}"],
         method: (ctx, input) => {
             console.log(input)
         }
     }
 }
2. Abstract language design
use my-module.js;
 if you == "developer" 
    then print "hi developer";
