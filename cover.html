Abstract language design

Puzzle is a programming language that has an abstract design for simple and powerful development

Define custom languages

Puzzle allows custom syntax definitions for domain-specific grammars.
syntax = {
     echo: {
         follow: ["{input}"],
         method: (ctx, input) => {
             console.log(input)
         }
     }
 }

 use my-syntax.js;
 echo "hello world";



puzzle.parse('print hello')
 
 <script type="text/x-puzzle">
     print hello;
 </script>
 
 puzzle run example.pz
