An abstract programming language,
that is versatile, extendable and runs on any platform

Modules 
 Use third-party modules
use https://abc.io/web-module.js;

create button with text "click me" and onclick "alert()"
Customize 
 Build a custom syntax
var my_syntax = {
  send: {
    follow: ["$email"],
    method: (ctx) => {...}
  },
  email: {...},
  ...
} 


 Use custom syntax

 ... or embedded in js

    print "hello world";
</script>
