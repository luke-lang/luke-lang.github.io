PUZZLE - Abstract Programming Language
Puzzle is a simple programming language that runs in javascript environments
and can be easily extended with custom or ready-to-use modules.
Hello World 
<script type="text/x-puzzle">
    print "hello world";
</script>
Use modules from any origin 
use https://abc.io/web-module.js;

create button with text "click me" and onclick "alert()"
Build your custom language, use it as module 
 1. Build a custom syntax using a simple js object
{
  send: {...},
  email: {...},
  to: {...},
  with: {...}
} 


 2. Use it
use my_syntax.js;

send email
  to peter@griffin.com
  with text "hi there";

print "email sent!";
Abstract language design 
 Use abstract syntax
render (
    <div id="123">Hello World</div>
);

create button inside 123 with text "click me" and onclick "alert()"
Runs in many ways 
 Standalone 
puzzle run myfile.pz;

 CLI
$ puzzle
$> print "hello world"


Embedded
puzzle.parse(`print "hello world";`)

 As script
<script type="text/x-puzzle">
    print "hello world";
</script>
