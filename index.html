Create a custom module


//mymodule.luke.js
syntax = {
  $: {
    'echo': {
      follow: ["{text}"],
      method: (ctx, text) => {
        console.log(text)
    }
  }
}

What is luke?
Luke is a programming platform for simple scripting, quick prototyping and automation, using simple syntax. It's goal is create a scripting language based upon a collaborative module ecosystem to make scripting super easy and accessible for developers and even non-developers.
